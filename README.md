# Synapse - Capstone Project

## What's the purpose of this project?

This capstone project's goal is to summarize the first milestone
of my coding journey.

This website is based simply on HTML and CSS (don't worry, tailwind & react
coming in the future 😉).

The focus was to build a clean & responsive website using Flexbox, Grid, etc.

## Live Demo

https://dekelpeer.github.io/Synapse---Capstone/

## Features

- Responsive Design
- Accessibility focues
- Gradient background
- Google fonts

- HTML5
- CSS3 (Flexbox, Grid, etc.)
